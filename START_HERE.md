## ⚡ Quick Start

### Prerequisites
- Node.js v14+ installed
- npm installed (comes with Node.js)

### Run the Application

```bash
# 1. Install dependencies
npm install

# 2. Start development server
npm run dev

# 3. Open browser to http://localhost:5173
```

**That's it! You're ready to use the application! 🎉**
